# Application Formulaire Tkinter

Application basique de formulaire développée avec Tkinter, démontrant l'utilisation des trois gestionnaires de layout : **pack**, **grid** et **place**.

## Fonctionnalités

- Formulaire de saisie avec deux champs (Nom et Prénom)
- Validation des champs avant affichage du message
- Affichage d'un message personnalisé "Bonjour [Nom] [Prénom]"
- Interface graphique simple et intuitive

## Gestionnaires de layout utilisés

- **pack** : Pour le label titre en haut de la fenêtre
- **grid** : Pour la grille de labels et champs de saisie (Nom/Prénom)
- **place** : Pour le bouton de validation en bas de la fenêtre

## Prérequis

- Python 3
- Tkinter

## Utilisation

1. Exécutez l'application :
   ```bash
   python3 app.py
   ```

2. Remplissez les champs :
   - Entrez votre nom
   - Entrez votre prénom

3. Cliquez sur le bouton "Valider" pour afficher le message de salutation.

- `app.py` : Fichier principal contenant l'application Tkinter
